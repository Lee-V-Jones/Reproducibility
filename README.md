# Reproducibility

[![DOI](https://zenodo.org/badge/974726625.svg)](https://doi.org/10.5281/zenodo.19448969)


This research project, part of a PhD by Lee Jones, examines statistical quality and reproducibility in health and biomedical research by focusing on linear regression models. This repository has all the materials required to create the manuscript “Challenges in the Computational Reproducibility of Linear Regression Analyses: An Empirical Study”   and  
"An Empirical Assessment of Inferential Reproducibility of Linear Regression in Health and Biomedical Research Papers"
by Lee Jones, Adrian Barnett, Gunter Hartel and Dimitrios Vagenas. 

A random sample of 95 papers was assessed for data availability. The first 20 papers with accessible data were then evaluated for computational reproducibility, with up to three models assessed per paper. Fourteen of these papers contained models that were at least partially reproducible and were therefore further assessed for inferential reproducibility.

The Paper folder contains the HTML report and Code for each of the 95 papers.
