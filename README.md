# Reproducibility

A random sample of 95 papers was assessed for data availability. The first 20 papers with accessible data were then evaluated for computational reproducibility, with up to three models assessed per paper. Fourteen of these papers contained models that were at least partially reproducible and were therefore further assessed for inferential reproducibility.

This file contains all the materials needed to create the computational inferential manuscripts. The Paper file also contains the HTML report and Code for each of the 95 papers.
